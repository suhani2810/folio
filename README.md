🚀 Folio - Personal Portfolio

Folio is a clean, modern, and responsive portfolio application built with
Flutter. It’s designed to showcase your work, skills, and experience across all
platforms (Web, Mobile, and Desktop) with a single codebase.

✨ Features

  - 📱 Fully Responsive: Adapts perfectly to Mobile, Tablet, and Desktop screens.
  - 🌓 Dark & Light Mode: Seamless theme switching for better user experience.
  - 🎨 Modern UI/UX: Built with Material Design 3 and custom animations.
  - 📧 Contact Form: Integrated for direct communication.
  - ⚡ Performance: Fast loading times and smooth 60 FPS animations.
  - 📁 Project Showcase: Interactive cards to display your GitHub repositories
    and live projects.

🛠️ Tech Stack

  - Framework: Flutter
  - Language: Dart
  - State Management: Provider (or Riverpod/Bloc)
  - Icons: Font Awesome, Material Icons
  - Animations: Flutter Animate / Lottie
  - Fonts: Google Fonts (e.g., Montserrat, Poppins)

🚀 Getting Started

Prerequisites

  - Flutter SDK installed (Guide)
  - Dart SDK installed

Installation

1.  Clone the repository
    git clone https://github.com/suhani2810/folio.git
2.  Navigate to the project folder
    cd folio
3.  Install dependencies
    flutter pub get
4.  Run the application
    flutter run

📂 Project Structure

lib/
├── configs/        # App themes, colors, and constants
├── core/           # Core utilities and shared widgets
├── models/         # Data models
├── providers/      # State management logic
├── sections/       # UI Sections (Home, About, Projects, Contact)
├── utils/          # Helper functions and responsive layout logic
└── main.dart       # Entry point
